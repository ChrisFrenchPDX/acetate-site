Acetate

A Node.js Static Site Generator

Extensiblity

Bring in custom data or write an extension to manipulate Acetate's build process.

Integration

Integrate Acetate into your existing toolchain and build process, whether it's Gulp, Grunt or just the command line.

Focus

Acetate only handles templating and building pages and lets you use whatever you want to build the rest of your application.
